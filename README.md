## 🎯 Sobre

Um jogo da forca multiplayer em Java usando socket.

## 🙋 Como usar

```
# Clone esse repositório
$ git clone https://github.com/oJPBarbosa/socket-hangman.git

# Entre na pasta do projeto
$ cd socket-hangman
```

- [Java](https://www.java.com/)
